GitHub Action Integration

Add MCP security scanning to your CI/CD pipeline in minutes. Results appear in the GitHub Security tab.

YAML .github/workflows/medusa.yml
name: MCP Security Scan
on: [push, pull_request]

jobs:
  scan:
    runs-on: ubuntu-latest
    permissions:
      security-events: write
    steps:
      - uses: actions/checkout@v4
      - uses: medusasecurity/medusa@v1
        with:
          server-command: 'npx -y your-mcp-server'
          fail-on: high
          # Results auto-upload to GitHub Security tab
